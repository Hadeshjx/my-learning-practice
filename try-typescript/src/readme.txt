面向对象

面向对象是程序中一个非常重要的思想。面向对象很简单，简而言之就是程序之中所有的操作都需要通过对象来完成。
例：
    操作浏览器需要使用window对象
    操作网页要使用document对象
    操作控制台要使用console对象

一切操作都要通过对象，也就是所谓的面向对象，那么对象到底是什么呢？这就要先说到程序是什么，计算机程序的本质就是对现实事物的抽象，抽象的反义词是具体，
比如：照片是一个对具体的人的抽象，汽车模型是对具体汽车的抽象等等。程序也是对事物的抽象，在程序中我们可以表示一个人、一条狗、一把枪、一颗子弹等等所有的事物。
一个事物到程序中就变成了一个对象。

在程序中所有的对象都被分成了两部分，即：数据和功能。数据在对象中被称为属性，而功能就被称为方法。

类
